1. Operator (연산자)
관계 연산자(비교연산자)
== : 같다
!= : 같지 않다
< > : 초과 , 미만
>= <= : 이상, 이하

논리 연산자
and : A && B : 두 조건식 모두 참이면 참
or : A || B : 두 조건식 중 하나라도 참이면 참

단항 연산자
not : !A : 조건식이 참이면 거짓, 거짓이면 참

삼항 연산자
조건식 ? 참일때의 값 : 거짓일때의 값

2. Control Statement (제어문)
프로그램의 흐름을 제어함.

2.1. 조건문
if : 조건식이 true이면 영역안의 코드가 실행됨
else if : 위의 조건식이 거짓이고 else if문의 조건식이 참이면 else if 영역안의 코드실행
else : 위의 조건식이 모두 거짓이면 else 영역안의 코드실행
=> else if, else문은 if 없이는 사용이 불가능하다.
switch : 변수가 case1, 2, ...와 동일하면 break로 빠져나옴.
switch문에서 case1, 2, ...가 변수와 같지 않을땐 default로 모두 다 같지 않을 때 실행할 문장을 입력함.
