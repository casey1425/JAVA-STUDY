0.1. 상속
class 부모클래스{
	//부모클래스의 멤버(필드, 메소드, 생성자)
}

class 자식클래스 extends 부모클래스{
	//자식클래스의 멤버
	//부모클래스의 멤버
}
자바에서 상속은 단일상속만 지원한다.
this : 객체 자기 자신의 참조값
this() : 자기자신의 생성자
super : 부모의 참조값
super() : 부모의 생성자

1. Ploymorphism(다형성)
- 하나의 것으로 여러 형태를 가지는 것.

1.1. 오버로딩(메소드, 생성자)
- 같은 클래스 내에서 생성자나 메소드를 같은 이름으로 매개변수의 타입이나 순서나 개수만 다르게 만드는 것.

1.2. 오버라이딩(메소드)
- 재정의하는것. 부모클래스에 정의한 메소드를 자식 클래스에서 수정해서 사용하고 싶다면 오버라이딩을 사용하면 됨.
이때 메소드명, 매개변수, 반환타입은 부모의 메소드와 동일해야한다.
구현부의 내용만 변경하면 된다.
오버라이딩 단축키 : alt + shift + s + v

2. 접근제한자(접근권한제어자, 접근자)
- (defalut) : 다른 패키지에서 접근 불가
- public : 모든 곳에서 접근 가능, 대표 클래스를 의미 할때 사용.
- protected : 다른 패키지에서 접근 불가, 자식은 허용
- private : 다른 클래스에서 접근 불가, 메소드로만 접근 가능(getter, setter)

접근제한자	적용대상				접근 가능 대상
public		클래스, 필드, 생성자, 메소드 	모든대상
protected	필드, 생성자, 메소드		같은 패키지, 자식 클래스
default		클래스, 필드, 생성자, 메소드	같은 패키지에 소속된 클래스
private		필드, 생성자, 메소드		클래스 내부

Getter : get필드명()
- 접근제한자 : public
- 리턴타입 : 필드의 리턴타입
- 메소드이름 : get+필드이름(첫문자는 대문자)
- 리턴값 : 필드값

Setter : set필드명()
- 접근 제한자 : public
- 리턴타입 : void
- 메소드이름 : set+필드이름(첫문자는 대문자)
- 매개변수타입 : 필드 타입

3. final 키워드
-final 접근제한자 타입변수명 = 값;//상수
-final/static 접근제한자 class 클래스명{}//상속이 불가능한 클래스
-final/static 접근제한자 리턴타입 메소드명(){}//오버라이딩이 불가능한 메소드
