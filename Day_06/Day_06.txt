0.2. 삼항연산자
조건식? 참일때의 값 : 거짓일때의 값;

1. Repetitive Statement (반복문)
for : for(초기식;조건식;증감식){
			실행할 문장;}
초기식 : 처음에 설정할 값 -> 변수
조건식 : 조건이 true면 영역안의 문장을 실행시키고 false면 for문 탈출
증감식 : 값을 얼마나 증감할지 정하는 식
for문의 알고리즘
1. 초기식
2. 조건식(조건식의 결과가 참이면 실행문, 조건식의 결과가 거짓이면 loop탈출)
3. 실행문이 끝나면 증감식
4. 증감식의 결과로 조건식을 다시 비교
5. 2번으로 진행

while
while(조건식){
	실행할 문장;}
조건식이 참인동안 반복한다
실생중에 조건식이 거짓이 될 수 있어야만 루프에 빠지지않는다.

for와 while의 차이점
for : 몇 번 반복할지 횟수를 알 때
while : 몇 번 반복할지 모를때

do~while : do{실행할문장;
	}while(조건식);

do~while은 while과 비슷하지만 최소 한번은 실행해야할때 사용

2. 기타 제어문
반목문에서 주로 사용함
break : 즉시 해당 중괄호 영역을 탈출함
continue : 즉시 다음 loop로 넘어감

3. 이중 for문

	}
